# Bushra Basharat — Portfolio

This is a modern, responsive portfolio built with React (JSX) and Tailwind CSS. It is a static front-end only project tailored for Bushra Basharat.

Getting started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Notes

- No backend included. The contact form is UI-only.
- Tailwind is configured in `tailwind.config.cjs`.
